# AI Governance System Mapping Examples

Status: Examples
Version: v0.1

This directory contains example system mapping records for the AI Governance Mapping System.

## Included examples

* `FORK_SYSTEM_MAPPING_RECORD_v0_1.md`  
  * Fork's own self-declared placement as the recomputable evidence-boundary layer.

* `GENERIC_RUNTIME_ANCHORING_SYSTEM_MAPPING_RECORD_v0_1.md`  
  * A generic lane example for runtime/public-log anchoring systems.

* `GENERIC_PRE_EXECUTION_GOVERNANCE_SYSTEM_MAPPING_RECORD_v0_1.md`  
  * A generic lane example for pre-execution permissioning or governance systems.

## Boundary

The generic examples are not mappings of named third-party systems.

They are lane examples only.

Named third-party systems should not be mapped as canonical records unless the relevant system owner provides or approves the mapping.
