# BC-10x Boundary Profiles

Classification: Informative  
Status: Draft v0.1

BC-10x profiles apply BC-100 to specific proof-surface seams.

Profiles are not integrations, adapters, bridges, or plugins. Profiles are declarative boundary specifications.

## Current Profiles

| Profile | Producer | Consumer | Status |
|---|---|---|---|
| BC-101 | AST-100 | FK-100 | Draft v0.1 |
| BC-102 | Reserved | FK-100 | Template |
| BC-103 | Reserved | FK-100 | Template |

## Rule

A profile MUST state whether it conforms to BC-100. A profile MUST NOT transfer authority or proof obligations between proof surfaces.
