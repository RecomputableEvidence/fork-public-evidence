Request the Executive Buyer Packet when the question becomes commercial fit, buyer pain, or pilot sponsorship.

Request the Technical Validation Packet when the question becomes packet mechanics, manifests, schemas, verifier behavior, PASS / FAIL / NOT_CHECKED states, tamper behavior, or technical reproducibility.

Request the Pilot Discovery Packet when the question becomes workflow suitability, source system inventory, evidence boundary mapping, stakeholder interviews, or pilot scoping.

## For qualified pilot opportunities

Use the Pilot-Ready Implementation Packet only after the workflow, source systems, sponsor, security expectations, and pilot objectives are sufficiently qualified.
