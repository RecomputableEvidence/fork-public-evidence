- Pilot scope

This package provides a bounded discovery scaffold for evaluating whether a defined AI-assisted workflow is suitable for Fork pilot scoping.

## Status

This package is a pilot discovery scaffold.

It is not a pilot-ready implementation package.

It is not a client-specific evidence boundary package.

It is not a production deployment claim.
