# Internal Working Materials Boundary

## Status

This directory is not part of Fork's released public-review surface.

Its contents are internal working materials, draft support materials, development notes, or operational-adjacent materials made visible in the repository for transparency and continuity.

## Citation boundary

Do not cite this directory as Fork's released public posture.

Do not treat files in this directory as:

- public doctrine;
- technical validation material;
- release package material;
- commercial offer material;
- client-ready material;
- production-readiness material;
- legal, compliance, audit, or security assurance material;
- evidence that Fork has made a claim not stated in a released package or public boundary document.

## Correct review path

For external review, use:

- `docs/REVIEWER_START_HERE_v0_1.md`
- `docs/PUBLIC_REVIEW_PACKAGE_INDEX_v0_1.md`
- `docs/VERIFICATION_COMMANDS_v0_1.md`
- `docs/FORK_NON_CLAIM_BOUNDARY_v0_1.md`

## Inclusion rule

A file in this directory becomes part of a bounded public-review surface only if a later release package, tag, pull request, or review request explicitly names that file and states the scope under which it is being reviewed.
