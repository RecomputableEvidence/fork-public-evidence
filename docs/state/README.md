# Fork Canonical Proof-Surface State

`FORK_PROOF_SURFACE_STATE_v0_1.json` is the machine-readable source for current implementation status, current public-review round, schema-enforcement scope, known limitations, and experimental admission state.

Run:

```bash
python tools/check_fork_proof_surface_state_v0_1.py --json --check-summary
```

The generated human-readable summary is:

- `FORK_PROOF_SURFACE_STATE_SUMMARY_v0_1.md`

A passing state check confirms only that the declared state is structurally valid, required paths exist, selected repository summaries remain synchronized, and known contradiction patterns are absent. It does not establish truth, compliance, legal sufficiency, safety, authorization, approval, certification, endorsement, production readiness, or institutional authority.
