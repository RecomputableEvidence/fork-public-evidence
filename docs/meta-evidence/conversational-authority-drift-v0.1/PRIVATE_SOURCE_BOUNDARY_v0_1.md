# Private Source Boundary v0.1

The raw conversation Markdown, companion TXT, and screenshots remain outside the public repository candidate.

The repository stores only:

- filenames;
- byte lengths;
- SHA-256 digests;
- source-role declarations;
- bounded claim dispositions;
- corrections, findings, and non-claims.

No public-repository file in this candidate is a substitute for the private source package. Reviewers requiring direct source inspection must receive the exact digest-bound package through a separately authorized channel.
