# CSH v0.1 Corpus

The six JSON files in this directory are preregistered scenario descriptors derived from existing Fork claim-inheritance simulation classes.

They define source claims, non-claims, expected boundary relationships, unresolved references, permitted transformations, and prohibited downstream outcomes.

Current status: descriptors present; byte-level corpus freeze not complete.
