# CSH Explicit Handoff-State Artifacts

Place scenario-specific `H = 1` handoff artifacts here before freeze.

Each artifact must satisfy:

- `schemas/cross_system_claim_handoff_state_artifact_v0_1.schema.json`;
- emitted claims and non-claims are explicit;
- authority and evidence references remain references;
- unresolved state is preserved;
- prohibited inferences are explicit;
- local revalidation obligations are explicit.

The control condition does not receive these artifacts.

This directory currently contains no frozen handoff artifact.
