# CSH Explicit Handoff-State Artifacts

This directory contains six scenario-specific `H = 1` handoff artifacts. Each artifact satisfies `schemas/cross_system_claim_handoff_state_artifact_v0_1.schema.json` and preserves emitted claims, non-claims, relationships, references, unresolved state, permitted narrowing, prohibited inferences, and local revalidation obligations.

The control condition receives no handoff artifact. These files remain unfrozen and do not authorize baseline execution.
