# Human Recomputation Sandbox — Boundary-State Interop v0.1.1

This sandbox invites human reviewers to independently inspect, run, annotate, and recompute the Boundary-State Interoperability v0.1.1 evidence package without inheriting authority from the original verifier, the packaged receipts, or Fork's public repository.

## Purpose

The sandbox is a bounded review surface for human recomputation. It preserves the evidence packet, reviewer instructions, exterior receipt, and candidate regression prompts so reviewers can exercise their own judgment over the boundary-state interop seam.

## Start here

1. Read HUMAN_RECOMPUTATION_SANDBOX_v0_1_1.md.
2. Run scripts/run_boundary_state_human_recomputation_v0_1_1.ps1 from the repo root with the evidence packet path.
3. Use HUMAN_REVIEWER_WORKSHEET_v0_1_1.md to record what you independently observed.
4. Treat exterior-receipts/INDEPENDENT_VERIFICATION_REPORT_v0_1_1.md as an exterior receipt, not as authority.
5. Review candidate payloads under candidate-regression-payloads/ only as triage candidates.

## Non-absorption posture

This branch does not ask reviewers to endorse Fork, the checker, or the original verification. It asks reviewers to recompute, inspect, articulate, and distinguish what reproduced from what remains open.
