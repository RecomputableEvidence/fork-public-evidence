# Ethics and Boundaries v0.1

Identifier: ETHICS_AND_BOUNDARIES_v0_1
Status: Draft

## Purpose

This note establishes the participation boundaries for the Exterior Observance Experiment v0.1.

## Participation

Participation should be voluntary.

Observers should understand that their comments may be preserved as bounded exterior observations only if permission is given or the communication context permits preservation.

Observers may request attribution, role-only attribution, anonymization, or exclusion from the public corpus.

## Non-endorsement

No observation should be treated as endorsement, certification, approval, validation, production-readiness assessment, legal conclusion, or compliance conclusion unless expressly stated and separately scoped.

Default endorsement status is none.

## Scope protection

If an observer states that work would require a paid or formal engagement, that boundary should be preserved and respected.

No further substantive architecture work should be solicited from that observer under an informal observation frame unless the boundary is clarified.

## Data minimization

Preserve only what is needed for architectural observance:

- observation text;
- scope;
- limitations;
- interaction type;
- non-endorsement;
- relevant architectural tags.

Avoid preserving unrelated personal information.

## Public use

Public use of observations should preserve:

- limitations;
- uncertainty;
- corrections;
- non-endorsement;
- attribution preference;
- the difference between observation and authority.
