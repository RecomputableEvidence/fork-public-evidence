# Exterior Observations

This surface preserves bounded exterior observations about Fork's evidence-boundary architecture. These artifacts are **observations**, not endorsements, reviews, certifications, or authority transfers.

Each observation is recorded as a machine-readable Markdown receipt with explicit metadata:

- identifier
- observer
- date
- interaction_type
- scope_requested
- scope_performed
- execution_performed
- endorsement
- limitations
- classification

The purpose of this corpus is to:

- Preserve what independent observers actually said, within their own scope and limitations.
- Make recurring architectural themes visible without treating them as consensus or proof of correctness.
- Provide a governed evidence surface that can be recomputed, analyzed, and extended over time.
