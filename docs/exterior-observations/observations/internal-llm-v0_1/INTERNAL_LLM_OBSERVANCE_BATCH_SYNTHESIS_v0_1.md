## 2. Recurring observation patterns

Across preserved internal LLM observations, several patterns appeared:

1. Boundary language is generally visible and disciplined.
2. Navigation and file access are unreliable for some LLM or automated observers.
3. Synthesis aggregation is the highest overclaim-risk stage.
4. Sandbox v0.2 should include observer-process failure cases in addition to artifact failure cases.
5. Truth-claims must remain explicit in the governing sandbox question.
6. interaction_type and classification require clearer separation.
7. Missing data and unknown data should not collapse into the same field value.

## 3. Apparatus refinements required
