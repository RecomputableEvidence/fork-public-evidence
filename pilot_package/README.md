# Pilot Package Directory Boundary

## Status

This root-level `pilot_package/` directory is an auxiliary controlled-pilot index area.

It is not the released pilot-discovery review package.

The released pilot-discovery review package is:

- `release_packages/FORK_PILOT_DISCOVERY_PACKET_v0_1/`

## Boundary

Do not treat this directory as:

- a released pilot offer;
- a commercial term sheet;
- a pricing document;
- a production-readiness claim;
- a deployment commitment;
- a client-specific suitability claim;
- a legal, compliance, audit, or security assurance artifact;
- a substitute for the released pilot discovery packet.

## Correct pilot discovery path

For pilot-discovery review, use:

- `release_packages/FORK_PILOT_DISCOVERY_PACKET_v0_1/README.md`
- `release_packages/FORK_PILOT_DISCOVERY_PACKET_v0_1/EVIDENCE_BOUNDARY_WORKSHEET.md`
- `release_packages/FORK_PILOT_DISCOVERY_PACKET_v0_1/SOURCE_SYSTEM_INVENTORY.md`
- `release_packages/FORK_PILOT_DISCOVERY_PACKET_v0_1/SECURITY_AND_DATA_HANDLING_QUESTIONS.md`
- `release_packages/FORK_PILOT_DISCOVERY_PACKET_v0_1/CLAIMS_AND_NON_CLAIMS.md`

## Interpretation rule

If any material in this directory appears to conflict with a released package under `release_packages/`, the released package governs the public-review interpretation unless a later release explicitly states otherwise.
